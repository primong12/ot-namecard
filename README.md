# HSproduce · 이효석아카데미 엘리트클럽 이미지 생성기

브라우저에서 바로 동작하는 단일 HTML 앱입니다. (인터넷 없이도 동작 / 폰트·라이브러리·모델·로고 내장)

- **엘리트클럽 프로필** 탭: 사진 + 명단으로 얼굴 1:1 크롭 프로필 카드 생성
- **엘리트 이름표** 탭: 109×70mm 이름표 (남색·골드 + 그로스=초록 / 밸류=빨강 팀 색), 개별 PNG·A4 PDF 저장
- **목걸이 명찰** 탭: 97×121mm 세로 명찰 (엘리트클럽 로고 + 기수 + 하단 기수 구분 띠)
- **스탠드 이름표** 탭: 가로 명찰 (목걸이 명찰과 동일 디자인, 사이즈 조정 예정)

기수 구분 띠 색: 1기 빨·2기 주·3기 노·4기 초·5기 파·6기 남·7기 보 (무지개 7색 순환) · 게스트 회색
