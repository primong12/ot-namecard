# OT 이름표 생성기 · 이효석아카데미 엘리트클럽

브라우저에서 바로 동작하는 단일 HTML 앱입니다. (인터넷 없이도 동작 / 폰트·라이브러리·모델 내장)

- **프로필 만들기** 탭: 사진 + 명단으로 얼굴 1:1 크롭 프로필 카드 생성
- **이름표** 탭: 90×60mm 이름표 생성 (남색·골드 베이스 + 그로스=초록 / 밸류=빨강 팀 색), 개별 PNG·A4 8장 PDF 저장

## 온라인 링크
GitHub Pages: https://primong12.github.io/ot-namecard/

## 빌드
원본은 `build/app.template.html`, 빌드는 `python3 build/build.py` (→ `index.html`, `ot_namecard_app.html` 생성).
